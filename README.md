# Fonoanas - Landing Page Dra. Ana Silva

Este projeto é uma landing page profissional desenvolvida para a **Dra. Ana Silva**, fonoaudióloga especializada em reabilitação infantil e adulta, voz e deglutição.

O objetivo principal da página é apresentar os serviços da clínica, fornecer informações sobre a experiência profissional da doutora e facilitar o agendamento de consultas através de um design moderno, responsivo e acolhedor.

## 🚀 Tecnologias Utilizadas

- **HTML5**: Estrutura semântica e acessível.
- **CSS3**: Estilização personalizada com foco em estética premium e responsividade.
- **Google Fonts**: Tipografia moderna (Outfit).
- **Vanilla JavaScript**: Para interações dinâmicas como o menu mobile.

## 📋 Funcionalidades

- **Design Responsivo**: Adaptado para dispositivos móveis, tablets e desktops.
- **Seções Informativas**:
  - **Hero**: Apresentação principal e chamada para ação (CTA).
  - **Sobre**: Detalhes sobre a formação e experiência da profissional.
  - **Tratamentos**: Listagem clara das especialidades atendidas (Terapia de Voz, Atraso de Linguagem, Disfagia, etc).
  - **Contato**: Formulário de contato funcional e informações de localização/WhatsApp.
- **Micro-interações**: Hover effects e navegação suave.

## 🛠️ Como Executar

Não há necessidade de compilação ou instalação de dependências pesadas, pois o projeto utiliza HTML e CSS puros.

1. Clone o repositório:
   ```bash
   git clone https://github.com/fabiotoniati/fonoanas.git
   ```
2. Abra o arquivo `index.html` em qualquer navegador moderno.
